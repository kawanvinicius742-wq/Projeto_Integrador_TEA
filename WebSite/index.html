Sobre nós

A DeepVision é uma equipe de desenvolvimento de software focada em desenvolver soluções inovadoras utilizando a visão computacional. Equipe formada durante o primeiro semestre na FATEC de Registro, no curso de Desenvolvimento de Software Multiplataforma.

Projetos

Descubra um dos nossos projetos em desenvolvimento com foco em visão computacional.

FaceTEA

Sistema desenvolvido para a terapia de crianças com Transtorno de Espectro Autista. Utiliza visão computacional para reconhecimento de expressões faciais em tempo real, além de gerar relatórios detalhados para análise do terapeuta.
